# Portfolio Website

A professional portfolio website for a Senior Laravel Developer & Product Manager, featuring a modern design, 3D elements, and technical blogs.

## Features

- **Modern UI/UX**: Clean, futuristic design with glassmorphism and dark mode.
- **3D Canvas**: Interactive 3D background using Three.js.
- **Technical Blogs**: Dedicated section for technical articles and insights.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Security**: Implemented security headers and best practices.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **3D Graphics**: Three.js
- **Icons**: Font Awesome
- **Fonts**: Syne, DM Mono

## Security Features

- **Security Headers**: X-Content-Type-Options, X-Frame-Options, Referrer-Policy, Permissions-Policy, Content-Security-Policy.
- **No Raw Mailto**: Contact forms use secure submission methods.
- **Robots Meta**: Optimized for search engines with `noarchive` tag.

## Getting Started

### Prerequisites

- A modern web browser.
- (Optional) A local development server for running PHP-related features if needed, though the site is primarily static.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. Open `index.html` in your browser.

## Usage

- Navigate through the sections: About, Skills, Experience, Projects, and Blogs.
- Use the contact form to get in touch.
- Click on blog titles to read technical articles.
